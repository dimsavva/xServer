The xServer is for hosting distributed applications. This software is for advanced users, please refer to the setup guide here.
Prerequisites
PostgreSQL
Collateral

In this version, there are three tiers of collateral. It is required that you have the x42 collateral to run one of the tiers on xServer.

Tier 1: 1000 x42
Tier 2: 20000 x42
Tier 3: 50000+ x42
